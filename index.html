Sharing options

— 2+ 명이 함께 쓰는 방법
OPTION 1 · 파일 동기화 (쉬움)
Google Drive / Dropbox / iCloud
dashboard.html 파일을 공유 폴더에 두고 각자 브라우저로 열기.
단: 데이터는 각자 브라우저 localStorage에 저장되므로
주기적으로 ⤓ Export → 공유 → ⤒ Import 필요.
OPTION 2 · 동시 작업 (권장)
GitHub Pages / Vercel 배포 + Firebase
현재 HTML을 정적 호스팅 + Firebase/Supabase 백엔드 연결.
실시간 공유 + 인증 + 여러 기기 동기화.
구축 시간: 1~2일 (개발 지원 필요).
OPTION 3 · 당장 시작하는 방법
역할 분담 + 주간 Export
한 명은 영업·리드 관리 (Sales · Intake 단계)
다른 한 명은 프로덕션 관리 (Production · QC · Delivery 단계)
금요일 주간 리포트 시 ⤓ Export → 병합 → 다음 주 ⤒ Import.
장점: 지금 즉시 가능, 자연스러운 업무 분담.
💡 추천 순서: Option 3로 시작 → 워크플로우 검증 → 한달 후 Option 2로 업그레이드.
